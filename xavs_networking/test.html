🚀 Quick Start

To install this module in your Cockpit environment:

sudo cp -r xavs_networking /usr/share/cockpit/
sudo chown -R root:root /usr/share/cockpit/xavs_networking
sudo chmod -R 644 /usr/share/cockpit/xavs_networking/*
sudo chmod 755 /usr/share/cockpit/xavs_networking
sudo systemctl restart cockpit

Then access via: https://your-server:9090

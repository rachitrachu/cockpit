Quick Start
  1. Configure network settings in the Network tab
  2. Set compute options in the Compute tab
  3. Configure storage and other services as needed
  4. Click Preview to review configuration
  5. Click Save Configuration to apply settings
Important Notes
  • Required fields are marked with *
  • A backup is automatically created when saving
  • Use Preview to review YAML before saving
  • Changes take effect after running ansible-playbook
